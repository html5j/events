HTMLとかCSSとかAPIとか
2011年12月のHTML5とか勉強会で好評を博した白石さんの「HTML5&API総まくり」、その2012年秋版です。今回はCSSも加え、時間の許す限りまくっていきます。

講演資料   講演映像

HTML5マークアップ珍プレー集
近年、個人や企業を問わず、HTML5でマークアップされたサイトが数多く登場しています。しかし、HTML5が浸透してから年月が浅いこともあり、有名な大企業のサイトですら、不適切なマークアップが見受けられます。本セッションでは、そのような不適切なマークアップを紹介し、どう直すべきかをみなさんと一緒に考察していきます。
羽田野 太巳 (有限会社 Futomi)
講演資料   講演映像

HTML5とIE10とWindows 8
IE9から本格化したHTML5の実装はまもなくリリースされるIE10でさらに広がるだけでなく、次期Windows OSであるWindows 8のアプリそのものもHTML5で構築できるようになります。このセッションではIE10で新たにサポートされるHTML5、Windows 8のアプリとHTML5との関係、そして、現在のマイクロソフトのHTML5との向き合い方について、ご紹介します。
春日井 良隆 (日本マイクロソフト株式会社)
講演資料   講演映像

HTML Web Platform
Android と iOS の両方にアプリを作るには WebView を使ったハイブリッドアプリが普通となり、Windows、Tizen、Firefox OS などでも HTML5 ベースでアプリケーション開発を行うようになりました。 これらの「HTML5 プラットフォーム」はどう違うのか？ Web 標準の進化と実装はこれからどうなっていくのか？ Web のプラットフォーム化に向けて多くの API 標準化・実装に取り組んでいる Mozilla ならではの視点で、Web 技術の現在と未来について、実際に目で見て分かる形でご説明します。
浅井 智也 (Mozilla Japan)

スマートデバイスの普及や高解像度ディスプレイの登場などによって、これまで のWebデザインの常識ももはや変わりつつあります。このセッションでは、閲覧 環境や仕様の変化にも柔軟に対応できる、ちょっと先の未来を見据えたWebデザ インの手法を紹介します。

実践Sass

WebIntentsにより拓かれる次のWeb
WebIntentsは、Androidインテンツに似た仕組みにより、Webアプリケーション間をユーザー主導で連携するAPIです。しかし、このAPIはアプリ連携のみならず、身の回りにある全てのデバイスをインタラクティブに利用出来る新たなマルチデバイスWebの可能性を秘めています。本講演では、WebIntentsとその可能性についてW3Cでの最新検討状況も交え紹介します。

Web最先端、エキスパートたちの視点から
HTML5、関連API、CSS3、WebSocketやSPDYといった新たな通信プロトコル、様々なフレームワーク、レスポンシブWebデザインなどの新たな制作手法、JavaScriptにコンパイルできる様々な言語など、Web技術の広がりと深まりはとどまるところを知りません。 本セッションでは、Google Developers ExpertやMicrosoft Most Valuable Professionalとして最先端で活躍するHTML5のエキスパートを集め、昨今のWebを縦横無尽に語ります。 モデレータにPublickeyの新野淳一氏、特別ゲストにグーグルの及川卓也氏を迎えた豪華メンバーでお届けします。

WebGL and Three.js
さまざまな新しい周辺技術が注目される中、Web の技術はグラフィックスの分野への強化もされています。そのひとつに WebGL という高度な 3D を扱う API があります。このセッションでは、主に THREE.js を通して WebGL を直感的に利用する方法を解説します。 このセッションを通し、JavaScript が苦手な人は WebGL の世界を知ることができるでしょう。また、JavaScript が得意な人は THREE.js を通して WebGL を利用する方法を知ることができるでしょう。

User Interface Design Pattern With CSS3
CSS3ではこれまで画像としてしか表現できなかったことを可能にする強力な仕様が追加されています。角丸やグラデーション、そしてアニメーションなどがその筆頭です。様々なプロパティの組み合わせを行うことでこれらの新しい仕様だけではなく、元々あったCSSのプロパティもより魅力的になるのです。 今回のセッションではその強力な組み合わせのプロパティを使ってより使いやすいUIの実践についてお話しします。
